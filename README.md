
# Space Invaders - Pygame

This is a basic functioning version of the arcade game Space Invaders build using Pyton and Pygame.

## Installation

Install pygame with pip

```bash
  pip install pygame
```
Run the game
```bash
  python -u "path/to/main.py" or
  python3 -u "path/to/main.py"

```
